# Medal Design – Mechanical Task

## Description
This repository contains a simple 3D medal created in Onshape for a mechanical design assignment.  
The medal features a 4mm keyring hole and a total extrusion depth of 2mm.  
The final model is exported as an STL file and is ready for 3D printing.
